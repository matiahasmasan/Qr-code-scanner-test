QR Scanner — Library Test

Comparing the top 3 JS libraries for a fidelity points webapp

html5-qrcode ★ 4.1k ~2 MB Built-in UI

Full-featured library that renders its own camera widget. Easiest integration — ideal for rapid prototyping. Large bundle size is its main downside.

✓ Scanned
nimiq/qr-scanner ★ 3.5k ~16 KB gz ZXing-based

Lightest option with the highest detection rate (2–3× better than competitors per benchmarks). No UI included — you bring your own. Best choice for production.

✓ Scanned
jsQR ★ 3.5k ~35 KB Pure JS

Low-level pure-JS decoder. You manage the camera stream and canvas yourself — maximum control, zero magic. Note: development has been dormant since 2021.

✓ Scanned
